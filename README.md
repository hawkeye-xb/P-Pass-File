# P-Pass-File

基于 WebRTC 个人的点对点文件管理跨平台应用。

## 为什么选择 P-Pass-File
### 简易：
- 不需要申请公网 IP、复杂的内网穿透、代理服务器等就能访问到你的文件。
- 不需要刷 Linux 系统、Docker 等环境，安装即可使用。
- 不需要额外的硬件，闲置设备随时安装随时使用。更无需为内存不够的 Mac 出门携带移动硬盘。

### 安全：
- 点对点传输，没有中间服务器。你的数据你自己掌握。

### 快速：
- 宽带的速度决定了文件的传输速度。
